\# ISEKAI XR - Server Monitor Tool



Lightweight PowerShell tool for real-time server monitoring and diagnostics.



Displays:

\- Ping latency (ICMP)

\- TCP port status

\## Purpose



This tool is designed for:



\- Server connectivity monitoring

\- Network diagnostics

\- Local and remote service testing

\- Development and debugging support



\---



\## Features



\- Real-time ping monitoring

\- TCP port status check

\- Color-coded latency display (in terminal)

\- Simple CLI usage (PowerShell + optional BAT launcher)

\- Custom IP / port via parameters

\- Works locally or remotely



\---



\## How to Use



\### Option 1 - Direct PowerShell



powershell -ExecutionPolicy Bypass -File "ping\_gameserv.ps1" -ip 127.0.0.1 -port 135



\---



\### Option 2 - Using BAT launcher



Run:



start.bat



Then enter:

\- IP / Host

\- Port



\---



\## Default Behavior



If no parameters are provided:



\- IP: 127.0.0.1

\- Port: 135



\---



\## Requirements



\- Windows OS

\- PowerShell 5.1 or higher

\- Network access to target host



\---



\## Files



\- ping\_gameserv.ps1 → main monitoring script  

\- start.bat → optional interactive launcher
